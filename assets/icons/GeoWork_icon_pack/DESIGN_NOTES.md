# GeoWork Visual Identity Notes

Style direction: professional geospatial AI software rather than a generic letter logo.

Core metaphor:
- Earth/globe = geography, GIS and remote sensing.
- Orbit/data route = GEE, automated data pipelines and model routing.
- Node workbench = AI Agent, tools, Skills, Plugins and MCP connectors.
- Contour/grid field = maps, rasters, vectors and spatial analysis.

Palette:
- Deep Navy `#071225`: trustworthy desktop software base.
- Geo Cyan `#8BFFE2`: spatial intelligence / GIS data.
- Signal Blue `#3AD9FF`: AI/data flow.
- Sand Gold `#F4D77E`: analysis/report output and premium accent.

Recommendation:
- Use `geowork-app-icon.svg` and 512/1024 PNG in marketing or README contexts where the full word is readable.
- Use `geowork-app-icon-compact.svg` or compact PNGs for tiny launcher sizes if the full-word icon becomes too small.

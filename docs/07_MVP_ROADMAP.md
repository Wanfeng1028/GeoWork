# GeoWork MVP 与路线图

## V0.1：最小可用 Demo

**目标**：跑通"NDVI 分析 + 报告生成"闭环。

**功能**：
- Electron 桌面壳
- Go Core Runtime
- Python Geo Worker
- 项目工作区
- 任务创建
- 计划生成
- SSE 日志
- GEE 脚本生成
- Word 报告生成
- 地图预览
- 成果列表

**验收任务**：
> 帮我完成成都市 2019-2024 年 NDVI 时序分析，并生成实验报告。

**验收标准**：
- 能创建项目
- 能创建任务
- 能生成计划
- SSE 显示执行过程
- Go 调 Python Worker
- 生成 GEE 脚本
- 生成 Word 实验报告
- 文件写入项目目录
- 底部日志显示工具调用记录

---

## V0.2：科研入口

**目标**：支持论文搜索和知识库。

**功能**：
- 论文搜索（OpenAlex 接入）
- PDF 导入
- 论文精读
- 文献综述矩阵
- 知识库
- Zotero 导入

**验收任务**：
> 帮我搜索近五年基于 Sentinel-2 的城市绿地变化检测论文，并整理研究空白和可行选题。

---

## V0.3：GIS 工程能力

**目标**：支持本地 GIS 处理。

**功能**：
- QGIS 环境检测
- QGIS Processing 调用
- GDAL 批处理
- 矢量/栅格预览
- 投影转换
- 空间分析
- 地图制图导出

**验收任务**：
> 帮我对这个 Shapefile 进行坐标转换、缓冲区分析，并生成专题地图。

---

## V0.4：Skill / Plugin / MCP

**目标**：支持技能、插件和 MCP 扩展。

**功能**：
- Skill 管理器
- Plugin 管理器
- 本地插件市场
- 权限审批
- MCP 管理器
- 官方 Skill 包
- 用户自定义 Skill

**验收任务**：
> 运行 NDVI 时序分析 Skill，并安装 OpenAlex 插件。

## V0.5：Code Mode

**目标**：支持代码编辑和 Git 集成。

**功能**：
- 代码编辑器

- Git 集成
- Diff 审查
- 运行测试
- Notebook 支持
- 任务回滚

**验收任务**：
> 帮我修改这个 GEE 脚本，添加云掩膜功能，并提交到 Git。

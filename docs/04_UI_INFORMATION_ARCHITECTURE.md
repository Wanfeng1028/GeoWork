# GeoWork 界面信息架构

## 1. 整体布局

GeoWork 使用"三栏工作台 + 顶部全局栏 + 底部执行面板"。

```
┌─────────────────────────────────────────────────────────┐
│ 顶部栏：项目切换 / 全局搜索 / 当前模型 / API 状态 / 用量  │
├──────┬──────────────────────────────┬───────────────────┤
│      │                              │                   │
│ 左侧 │       中间区                 │    右侧栏         │
│ 导航 │  对话/计划/执行/成果          │  地图/论文/数据   │
│      │                              │    参数/Diff      │
│      │                              │                   │
├──────┴──────────────────────────────┴───────────────────┤
│ 底部栏：任务日志 / 工具调用 / 终端 / GEE / QGIS / Python │
└─────────────────────────────────────────────────────────┘
```

### 顶部栏

- 项目切换下拉
- 全局搜索
- 当前模型显示
- API 状态指示
- 用量快速查看
- 通知中心
- 个人头像

### 左侧栏

- 主导航（16 个一级模块）
- 工作区安全模式指示

### 中间区

- 三个 Tab：对话、计划、成果
- 计划状态：待执行、执行中、等待确认、已完成、失败、已跳过
- 成果列表：GEE 脚本、Python 脚本、GeoTIFF、CSV、图表 PNG、交互地图 HTML、DOCX 报告、PPT、任务日志

### 右侧栏

- 地图预览
- 论文摘要
- 数据详情
- 参数面板
- Diff 视图
- 插件面板

### 底部栏

- 任务日志
- 工具调用记录
- 终端
- GEE 状态
- QGIS 状态
- Python 输出

| 8 | 专家 | TeamOutlined | 12 个内置专家角色 |
| 9 | 技能 | ExperimentOutlined | Skill 包管理、运行、自定义 |
| 10 | 自动化 | ToolOutlined | 触发器、条件、动作规则配置 |

| 12 | 扩展 | AppstoreOutlined | 工作流编排、Eino 适配器 |
| 13 | 插件市场 | CloudServerOutlined | 本地插件管理、权限审批 |

## 4. 工作台

### 三个 Tab

| Tab | 内容 |
|-----|------|
| 对话 | 用户输入、AI 回复、工具调用结果 |
| 计划 | 步骤列表、状态指示、进度条 |
| 成果 | 产物列表、预览、下载 |

### 计划状态

| 状态 | 说明 |
|------|------|
| 待执行 | 灰色，等待执行 |
| 执行中 | 蓝色，正在执行 |
| 等待确认 | 黄色，等待用户确认 |
| 已完成 | 绿色，执行成功 |
| 失败 | 红色，执行失败 |
| 已跳过 | 灰色，跳过执行 |

### 成果列表

- GEE 脚本
- Python 脚本
- GeoTIFF
- CSV
- 图表 PNG
- 交互地图 HTML
- DOCX 报告
- PPT
- 任务日志

## 5. 专家页面

### 专家卡片展示

### 内置专家列表

| 专家 | 角色 |
|------|------|
| 总控专家 | 任务规划、步骤编排、结果汇总 |
| 论文专家 | 文献搜索、综述生成、引用管理 |
| 数据专家 | 数据源推荐、数据下载、数据质量检查 |
| GEE 专家 | Google Earth Engine 脚本生成 |
| QGIS 专家 | QGIS Processing 调用 |
| 遥感分析专家 | 指数计算、分类、变化检测 |
| GIS 工程专家 | 坐标转换、投影、空间分析 |
| 地图制图专家 | 专题地图、图例、比例尺、导出布局 |
| 实验报告专家 | 实验步骤、数据、图表、结论生成 |
| 论文写作专家 | 论文大纲、初稿、修改、格式 |
| 质量检查专家 | 可复现性检查、数据溯源、逻辑验证 |
| 代码审查专家 | 代码质量、性能优化、安全审查 |
